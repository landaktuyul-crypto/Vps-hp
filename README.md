# Virtual Keyboard

This is a small web-based virtual keyboard you can open in a browser.

Files added:
- [index.html](index.html)
- [style.css](style.css)
- [keyboard.js](keyboard.js)

Usage:

- Open [index.html](index.html) directly in a browser (double-click or open file).
- Or run a simple local HTTP server and open http://localhost:8000:

```bash
cd /workspaces/Vps-hp
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Click the textarea, then click keys on the virtual keyboard below it to type.
